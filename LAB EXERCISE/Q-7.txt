Explanation:
-------------
1.System Software:-
--------------------
Operating System: Manages hardware and provides a platform for applications (e.g., Windows, macOS).
File Manager: Allows you to manage files and folders (e.g., File Explorer, Finder).

2.Application Software:-
-------------------------
Web Browser: Used for browsing the internet (e.g., Chrome, Firefox).
Text Editor: Used for writing and editing text or code (e.g., Notepad, Sublime Text).
Antivirus Software: Protects your computer from malware (e.g., Norton, McAfee).
